# 17.11.25.nb

**Description:**
A repeat of the caesar cipher test, using a local qwen 3 model in place of gemma3.

## Code

```mathematica
temps = Range[0, 2, 0.1];

Q34B = Table[
    Table[
        LLMSynthesize["here is a coded message:'N ymnsp ymjwjktwj N fr', decipher it please.", 
            LLMEvaluator -> LLMConfiguration[Association[
                "Model" -> {"Ollama", "qwen3:4b"}, 
                "Temperature" -> i
            ]]
        ],
        50
    ],
    {i, temps}
];

Q34BT = Transpose[Q34B];

Export["OLQWEN34B.xlsx", "Q34B" -> Q34BT, "XLSX"]
```
